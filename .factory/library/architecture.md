# Architecture

This part of the mission focuses on a Python data ingestion pipeline `import_user_data.py`.
It reads JSON output generated by chuumai-tools Docker containers, and upserts it into the `user_scores` table in PostgreSQL.
It is required to keep only the latest 5 snapshots per game by pruning older snapshots.
It uses `asyncpg` for database connection, reusing existing db utilities where appropriate.
